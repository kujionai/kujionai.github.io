.to_a
.to_a

Returns array of all elements (and actually tries to convert the parameter into an array).

"bagel".to_a #=> ["bagel"]
.include?
.include?

Returns true if the given object is present in self (that is, if any object == anObject), and false otherwise.

bagels = [ "plain", "pumpernickel", "everything", "sesame" ]
bagels.include?("plain") #=> true
bagels.include?("jazz hands") #=> false
.map
.map

Returns an array with the results from block.

bagels = [ "plain", "pumpernickel", "everything", "sesame" ]
bagels.map! {|x| x + "!" }
bagels #=>  [ "plain!", "pumpernickel!", "everything!", "sesame!" ]
.sort_by
.sort_by

Returns array sorted by the return value of the block.

%w{plain pumpernickel everything sesame}.sort_by { |word| word.length}
#=> ["everything", "plain", "pumpernickel", "sesame"]
.each
.each

Goes through each item in the object you called it on and passes it to the block that you specified.

bagels = [ "plain", "pumpernickel", "everything", "sesame" ]
bagels.each {|x| x + "! " }
# prints: plain! pumpernickel! everything! sesame!
bagels #=>  [ "plain", "pumpernickel", "everything", "sesame" ]
.reverse_each
.reverse_each

Yields all elements in reverse order.

bagels = [ "plain", "pumpernickel", "everything", "sesame" ]
bagels.map! {|x| x + "!" }
bagels #=>  [ "plain!", "pumpernickel!", "everything!", "sesame!" ]

.select
.select

Runs the block on every item of your object and returns a new object that contains only those items for which the original block returned true.

bagels = [ "plain", "pumpernickel", "everything", "sesame" ]
bagels.map! {|x| x + "!" }
bagels #=>  [ "plain!", "pumpernickel!", "everything!", "sesame!" ]
