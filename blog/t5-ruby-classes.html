Ruby Classes
Greeting the wildlife with ordered expectations.

Not long after moving a stone's throw South of Brooklyn's Prospect Park about six months ago, I fell in love with the neighborhood's WILDLIFE. We're talking squirrels. Sparrows. And to my purest delight, we've got cats on cats on cats EVERYWHERE.

To Ruby-document some of my favorite feline neighbors, I've created the following class called Neighborhood_Cat. 

class Neighborhood_Cat
  def initialize(location, favorite_activity)
    @location = location
    @favorite_activity = favorite_activity
  end
  def hang_out
    puts "My favorite hangout spot is the #{@location}! Next time we meet, I'll definitely #{@favorite_activity}."
  end
  def meow
    puts "Meow! Meow!"
  end
end

Classes are perfect for grouping behaviors (methods) into convenient bundles that allow you to effortlessly create many objects that behave in basically the same way. By creating the class Neighborhood_Cat, I can now create objects that are instances of this class - different neightborhood cats that all meow and hang_out (these are the methods bundled in the class), but have different locations and favorite_activitys (these are the arguments passed into the initialize method).

Each Neighborhood cat has a favorite location and an expected favorite_activity, so I'll make those the arguments for Neighborhood_Cat's initialize method (initialize gets called each time we create a new instance of a class).

brown_cat = Neighborhood_Cat.new("corner deli checkout counter", "stay asleep while you try to get my attention")
brown_cat.hang_out
brown_cat.meow

black_cat = Neighborhood_Cat.new("overgrown lot next to the Catholic School", "run away and then look back at you like I want you to chase me")
black_cat.hang_out
black_cat.meow

grey_cat = Neighborhood_Cat.new("new duplex close to the subway stop", "butt my head against your hand")
grey_cat.hang_out
grey_cat.meow

Sweet! We created new instances of our Neighborhood_cats class and called the methods on each instance. Here's what we'll get when we run this in our terminal:

=> My favorite hangout spot is the corner deli checkout counter! Next time we meet, I'll definitely stay asleep while you try to get my attention.
=> Meow! Meow!
=> My favorite hangout spot is the overgrown lot next to the Catholic School! Next time we meet, I'll definitely run away and then look back at you like I want you to chase me.
=> Meow! Meow!
=> My favorite hangout spot is the new duplex close to the subway stop! Next time we meet, I'll definitely butt my head against your hand.
=> Meow! Meow!

Awesome! Keeping all those cats straight is pretty simple with a class. Don't forget to greet your furry neighbors!
